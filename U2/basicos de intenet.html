BASICOS DEL INTERNET

¿Qué es Internet?

Internet es una red global de computadoras interconectadas que permite la comunicación y el intercambio de información.

Funciona a través de una serie de protocolos estandarizados, como el Protocolo de Internet (IP), que permiten que dispositivos de todo el mundo se conecten y se comuniquen entre sí, formando una vasta red de datos.

Tipos de Proveedores de Internet y nombres

Un Proveedor de Servicios de Internet (ISP) es una empresa que proporciona acceso a Internet a sus clientes.

Existen varios tipos principales de proveedores:

  • ISP de banda ancha: Ofrecen conexiones de alta velocidad a través de tecnologías como la fibra óptica (Telmex, Totalplay), el cable coaxial (Izzi, Megacable) y el DSL.
  • ISP inalámbricos: Proporcionan acceso a Internet a través de tecnologías inalámbricas, como redes móviles (Telcel, AT&T) o Internet fijo inalámbrico (WISP), que usan ondas de radio.
  • ISP satelitales: Ofrecen conectividad a través de satélites en órbita, ideal para áreas rurales o remotas donde otras opciones no están disponibles (Starlink, Viasat).

¿Qué es WWW?

WWW son las siglas de World Wide Web, un sistema de documentos de hipertexto interconectados a los que se accede a través de Internet.

A menudo se confunde con Internet, pero la WWW es solo una parte de ella. Es el sistema de páginas web que usas a diario, donde los documentos están vinculados entre sí mediante enlaces, y se visualizan a través de un navegador web.

¿Qué es carpeta local y carpeta remota?

Una carpeta local es un directorio de archivos que se encuentra en tu propio dispositivo (computadora, teléfono o tablet), mientras que una carpeta remota se almacena en un servidor o computadora diferente a la tuya y se accede a través de una red, como Internet.

Por ejemplo, los archivos en el disco duro de tu computadora están en una carpeta local, mientras que los archivos en Google Drive o Dropbox están en una carpeta remota.

¿Qué es FTP?

FTP son las siglas de File Transfer Protocol, un protocolo de red utilizado para la transferencia de archivos entre un cliente y un servidor en una red de computadoras.

Se usa comúnmente para subir o descargar archivos a o desde un servidor web, permitiendo a los desarrolladores o administradores de sitios web gestionar el contenido de su página.

¿Qué es experiencia de usuario?

La experiencia de usuario (UX) se refiere a todas las emociones, actitudes y percepciones que una persona tiene al usar un producto, sistema o servicio.

Se centra en hacer que la interacción sea lo más eficiente, agradable y fácil posible. No se trata solo del diseño visual, sino de cómo se siente el usuario al navegar por una página web, usar una aplicación o interactuar con un producto.

¿Qué es interfaz de usuario?

La interfaz de usuario (UI) es el medio a través del cual una persona interactúa con un dispositivo o aplicación.

Es la parte visible y tangible del diseño. Incluye elementos como botones, menús, íconos, tipografía y colores. La UI se enfoca en el aspecto visual y el diseño de los elementos con los que el usuario interactúa.

¿Qué es un DNS?

DNS son las siglas de Domain Name System (Sistema de Nombres de Dominio), un sistema que traduce los nombres de dominio legibles para humanos (como "www.google.com") en direcciones IP numéricas que las computadoras usan para identificarse en la red (como "172.217.1.1").

Funciona como una "agenda telefónica" de Internet, permitiendo que tu navegador encuentre el servidor correcto para la página web que deseas visitar.

¿Qué es un Hosting?

El hosting (o alojamiento web) es un servicio que permite a una persona o empresa publicar una página web o aplicación en Internet.

Un proveedor de hosting alquila espacio en sus servidores, que son computadoras potentes y siempre conectadas, para almacenar los archivos y datos de tu sitio web, haciendo que esté disponible para que cualquiera lo visite.

¿Qué es un Full Stack Designer?

Un Full Stack Designer es un profesional del diseño que tiene habilidades tanto en el diseño de la experiencia de usuario (UX) como en el diseño de la interfaz de usuario (UI), y además tiene conocimientos o experiencia en desarrollo web.

Este perfil híbrido le permite trabajar en todo el proceso de un proyecto digital, desde la conceptualización y el diseño visual hasta la implementación técnica, entendiendo las limitaciones y posibilidades de cada etapa.
